# Trend Sonar MVP - 核心玩法设计文档

## 📖 概述

Trend Sonar 是一款基于"趋势预测投资"概念的时尚发现应用，将时尚潮流观察与游戏化风险投资相结合，让用户通过预测趋势走向获得收益和成就感。

## 🎮 核心玩法流程

### 1. 趋势发现与提交
- **用户发现**：用户在日常生活中观察到新的时尚趋势（服装、配饰、风格等）
- **提交申请**：通过"发现新趋势"页面提交趋势信息，包括名称、类别、描述、灵感来源
- **社区审核**：提交的趋势进入待审核状态，模拟社区投票机制
- **趋势上线**：审核通过的趋势以"小众"状态进入雷达系统

### 2. 趋势投资预测
- **雷达扫描**：用户在个性化雷达中发现感兴趣的小众趋势
- **风险投资**：选择预测目标（先锋区/主流区）并投注声纳币
- **信心设定**：设置预测信心指数（0-100%），影响收益倍数
- **等待验证**：系统在7天内验证预测结果（演示版2秒）

### 3. 收益与激励系统
- **预测成功**：获得 `投注金额 × 难度倍数 × 信心奖励` 的声纳币
- **积分累积**：成功预测获得积分，用于等级提升和成就解锁
- **连击奖励**：连续预测成功获得额外奖励
- **发现奖励**：提交的趋势变火爆时，早期支持者获得大额奖励

## 🎯 目标用户群体

### 核心用户：时尚敏感的年轻群体（18-30岁）
- **穿搭博主/KOL**：寻找下一个爆款内容话题
- **时尚爱好者**：享受发现和预测潮流的成就感
- **大学生群体**：有时间关注潮流，喜欢社交分享
- **潮流消费者**：希望提前布局，避免"撞衫"尴尬

### 使用场景
- **内容创作**：博主寻找独特选题和穿搭灵感
- **购物决策**：提前布局可能火爆的单品
- **社交炫耀**：展示自己的"时尚嗅觉"
- **娱乐消遣**：类似炒股的预测游戏体验

## ⚙️ 系统机制设计

### 趋势定位算法
系统通过以下维度确定趋势在雷达中的位置：

#### 热度分区规则
- **小众区（蓝色，距离中心远）**：热度 < 60
  - 新提交的趋势起始区域
  - 只有少数敏锐用户关注
  
- **先锋区（黄色，距离中心中等）**：热度 60-79  
  - 开始被更多用户关注
  - KOL开始尝试的阶段
  
- **主流区（红色，距离中心近）**：热度 ≥ 80
  - 大众普遍接受
  - 商业化程度最高

#### 热度计算因子
1. **用户关注度**：预测数量、查看次数
2. **社交传播**：分享数、讨论热度
3. **商业化指标**：相关商品销量、品牌采用度
4. **时间衰减**：过时的趋势热度自然下降
5. **外部数据**：社交媒体提及量、搜索趋势

### 动态更新机制
- **实时监控**：每15秒更新趋势热度（演示版）
- **自然演化**：热度有随机波动，模拟真实市场
- **用户影响**：预测行为会轻微影响趋势热度
- **生命周期**：趋势有兴起、爆发、衰落的完整周期

## 🔄 商业扩展性

### 积分经济系统
- **会员体系**：积分兑换VIP权益（优先预测、专属内容）
- **抽奖营销**：积分参与品牌联名抽奖活动
- **购物优惠**：积分兑换合作电商平台优惠券
- **实物奖励**：高积分用户获得限量时尚单品

### 商业合作模式
- **品牌植入**：时尚品牌赞助特定趋势，提升曝光
- **KOL合作**：与时尚博主合作，提升平台权威性
- **电商导购**：趋势页面关联商品购买链接
- **数据服务**：向时尚品牌出售趋势预测数据

### 社区生态扩展
- **UGC内容**：用户上传穿搭照片，AI分析风格标签
- **社交功能**：关注预测准确的"时尚先知"
- **挑战赛事**：定期举办预测大赛，增强用户粘性
- **线下活动**：组织时尚沙龙，连接线上线下

## 🎲 风险投资玩法核心

### 投注机制
- **声纳币系统**：虚拟货币，可充值或通过预测获得
- **多级投注**：5/10/20/50 声纳币不同风险等级
- **收益计算**：
  ```
  收益 = 投注金额 × 基础倍数 × 信心加成
  
  基础倍数：
  - 小众→先锋：3倍
  - 小众→主流：5倍  
  - 先锋→主流：2倍
  
  信心加成：
  - 信心>90%：+1倍
  - 信心>80%：+0.5倍
  ```

### 个性化推荐
- **风格匹配**：基于用户风格偏好过滤趋势
- **兼容性评分**：计算趋势与用户风格的匹配度
- **智能推送**：优先推荐高匹配度的投资机会

## 🏆 成就与等级系统

### 等级晋升
- **Lv.1 新手探索者**：完成首次预测
- **Lv.2 趋势观察员**：成功预测5次
- **Lv.3 时尚预测师**：积分达到300
- **Lv.4 潮流引领者**：积分达到600  
- **Lv.5 时尚大师**：积分达到1000
- **Lv.6+ 传奇预言家**：顶级玩家

### 特殊成就
- **趋势捕手**：成功预测10个趋势
- **先知先觉**：提前2周预测成功
- **社区贡献**：提交5个被采纳的趋势
- **连击王者**：连续预测成功10次
- **雷达专家**：使用应用超过30天

## 📊 数据闭环

### 用户行为数据
- **预测偏好**：分析用户喜欢预测的趋势类型
- **风险特征**：用户的投注风格和风险承受度
- **时机把握**：用户对趋势时机的判断能力
- **社交影响**：用户在社区中的影响力

### 商业价值输出
- **趋势报告**：定期输出市场趋势分析报告
- **用户画像**：为合作品牌提供精准用户洞察
- **预测模型**：不断优化的时尚趋势预测算法
- **营销策略**：基于数据的时尚营销建议
